Тебе дан текст и триплет, извлеченные из этого текста, для формирования триплета из графа знаний Wikidata.
В графе знаний каждый триплет содержит две сущности (субъект и объект) и одно отношение, которое связывает эти субъект и объект.

Твоя задача — сгенерировать определение для отношения из предоставленного триплета, которое использовалось в предоставленном текста.
Определение должно соответствовать контексту предоставленного текста.

Входной текст, триплет и отношение озаглавлены как «Text:», «Triplet:» и «Relation:» соответственно.
Сгенерируйте вывод в формате JSON с отношением в качестве ключа и определением в качестве значения.
