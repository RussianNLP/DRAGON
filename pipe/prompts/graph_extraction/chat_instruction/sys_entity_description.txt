Тебе дан текст и триплеты, извлеченные из этого текста, для формирования триплета из графа знаний Wikidata.
В графе знаний каждый триплет содержит две сущности (субъект и объект) и одно отношение, которое связывает эти субъект и объект.

Твоя задача — сгенерировать определение для сущности из предоставленного триплета, которое использовалось в предоставленном текста.
Определение должно соответствовать контексту предоставленного текста.

Входной текст, триплет и сущность озаглавлены как «Text:», «Triplet:» и «Entity:» соответственно.
Предоставь вывод в формате JSON с сущностью в качестве ключа и ее определением в качестве значения.
