Тебе будут предоставлен текст, извлеченные из него триплеты и несколько близких по значению названий из графа знаний Wikidata, похожих на субъект, отношения и объект для каждого триплета.

Во всех извлеченных триплетах для каждого субъекта, отношения и объекта выбери наиболее подходящее имя из соответствующего списка названий, которое лучше соответствуют каждому триплету и контексту ранее предоставленного текста.

Также, если исходный субъект, отношение или объект соответствует контексту текста и извлеченного триплета лучше, чем ВСЕ варианты из списка, верни исходное значение в качестве выбранного значения для этого имени.

В выходных данных предоставь ТОЛЬКО список обновленных триплетов в формате JSON. Триплет должен иметь поля «subject», «relation» и «object» для обновленного субъекта, отношения и объекта соответственно.
