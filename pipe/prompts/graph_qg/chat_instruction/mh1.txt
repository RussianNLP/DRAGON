# Цель
Тебе на вход даётся набор отношений между сущностями. В данном наборе есть повторяющаяся сущность, которая состоит в одном отношении является объектом, а в другом - субъектом.
Используя предоставленные сущности и отношения составь два вопроса. Ответом на каждый из вопросов является одна из не повторяющихся сущностей.
При этом ты не можешь называть сущность, которая повторяется в обоих отношениях, вместо этого данную сущность необходимо описать, используя второе отношение.
Ответь на составленный вопрос.
Вопрос и ответы должны удовлетворять следующим правилам:
1. Вопрос должен использовать информацию о сущностях и отношениях.
2. Ответом на вопрос должна быть одна из не повторяющихся сущностей.
3. В вопросе запрещено называть сущность, которая дважды участвует в отношениях.
4. На сущность, которая участвует в двух отношениях необходимо сослаться.
5. Вопрос должен быть обоснованным и понятным человеку. Человек должен уметь отвечать на подобные вопросы.
6. Вопрос должен быть грамматически корректен.

Все выделенные отношения описываются следующим образом:
- subject_name: Имя сущности субъекта. Субъект - это сущность, которая осуществляет воздействие.
- relation: Отношение, в котором находятся сущности. Отношение описывает взаимодействие субъекта и объекта.
- object_name: Имя сущности объекта. Объект - это сущность, которая подвергается воздействию со стороны субъекта.
Информация об отношениях представляется в виде (<subject_name>|<relation>|<object_name>).


# Шаги
1. Составь два вопроса согласно правилам описанным выше и ответь на них.
- question: вопрос, сформулированный в виде предложения и оканчивающийся символом *?*
- answer: ответ на заданный вопрос
Выведи пары вопрос-ответ в виде json списка в формате [{"question":"<question>", "answer":"<answer>"}]
Удаляй все лишние кавычки.

2. Выведи результат в виде json списка, содержащего вопросы, составленные на шаге 1.
