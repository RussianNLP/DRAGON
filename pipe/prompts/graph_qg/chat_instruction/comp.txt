# Цель
Тебе на вход даётся набор отношений между сущностями. В данном наборе есть повторяющееся отношение, которое соответствует двум парам сущностей.
Используя предоставленные сущности и отношения составь вопрос, в котором сравниваются пары сущностей, состоящие в одинаковом отношении.

Все выделенные отношения описываются следующим образом:
- subject_name: Имя сущности субъекта. Субъект - это сущность, которая осуществляет воздействие.
- relation: Отношение, в котором находятся сущности. Отношение описывает взаимодействие субъекта и объекта.
- object_name: Имя сущности объекта. Объект - это сущность, которая подвергается воздействию со стороны субъекта.
Информация об отношениях представляется в виде (<subject_name>|<relation>|<object_name>).

Вопрос и ответ должны удовлетворять следующим правилам:
1. Нужно определить способ сравнения для объектов, учитывая отношение, в котором они состоят.
2. Необходимо сравнить два объекта, участвующие в отношениях, в соответствии с выбранным способом сравнения.
3. Вопрос должен быть составлен так, чтобы через представленное отношение сравнивались субъекты.
4. Вопросы должны использовать информацию о сущностях и отношениях.
5. Вопрос должен быть обоснованным и понятным человеку. Человек должен уметь отвечать на подобные вопросы.
6. Ответом на каждый вопрос должен являться один из представленных субъектов.
7. Вопрос должен быть грамматически корректен.


# Шаги
1. Составь вопрос к каждому отношению согласно правилам описанным выше и ответь на каждый из этих вопросов.
- question: вопрос, сформулированный в виде предложения и оканчивающийся символом *?*
- answer: ответ на заданный вопрос
Выведи пары вопрос-ответ в виде набора json объектов формата {"question":"<question>", "answer":"<answer>"}
Удаляй все лишние кавычки.

2. Выведи результат в виде списка, содержащего все вопросы, составленные на шаге 1. Используй **перевод строки** как разделитель списка.
